# 项目目标

## 概述

**edgetunnel** 是一个基于 Cloudflare Workers/Pages 平台的边缘计算隧道解密方案。它利用 CF 的全球边缘网络，实现高效的流量转发、协议解密和代理功能。

## 核心目标

1. **协议代理隧道**
   - 支持 VLESS、Trojan、Shadowsocks 三种主流代理协议
   - 通过 WebSocket（WS, XHTTP）、gRPC 传输层承载流量
   - 实现完整的 TLS 客户端握手（TLS 1.2/1.3），支持 ECDHE 密钥交换

2. **链式代理能力**
   - SOCKS5、HTTP/HTTPS、TURN、SSTP 多级代理链
   - ProxyIP 反代机制（通过 TXT/DNS 记录反代解析）
   - 全局/白名单代理模式

3. **订阅系统**
   - 自动生成适用于 Clash、Sing-box、Surge、Quantumult X、Loon 等客户端的订阅配置
   - 内置订阅转换后端对接
   - 优选IP/优选API机制

4. **管理面板**
   - 基于 KV 存储的配置持久化
   - 实时配置修改（config.json, cf.json, tg.json 等）
   - 日志查看、Cloudflare 用量查询
   - 代理节点连通性检测

5. **性能特性**
   - 并发拨号优化（TCP并发拨号数控制）
   - 预加载竞速拨号（DNS 解析后并发连接最快 IP）
   - 上行合包发送（减少小包发送次数）
   - 0RTT 支持

## 部署方式

- **Cloudflare Workers**: 直接粘贴代码
- **Cloudflare Pages**: 上传 ZIP 部署（推荐方式）

## 依赖

- Cloudflare KV 命名空间（绑定名 `KV`）
- 环境变量 `ADMIN`（管理员密码）
